sys-sage Documentation

This is the documentation of sys-sage.

sys-sage is a software library with a C++ API. The main goal of the library is to to store, update, and provide all relevant information about hardware topology, dynamic system state/configuration, system capabilities, and other data related to HW from different data sources logically connected to each other.
